# 《桥想》

## 简介

《桥想》是我个人开发的 **2025卡赢杯开源独立游戏开发赛作品**。  
游戏以主角 **松乔** 的视角展开，讲述其在心理医生的引导下，通过"关于桥的想象疗法"逐步缓解自身的创作焦虑与同伴压力的故事。

本作不仅是一个互动叙事游戏，也是对广大创作者群体中普遍存在的 **创作焦虑** 现象的一次探讨与对话。

## 环境要求

- **操作系统**：Windows 10 或更高版本
- **无需额外安装环境**，下载即可运行

## 玩法说明

《桥想》是一款 **线性视觉互动小说** 为主体的游戏，操作简单直观：

- 主要通过 **鼠标点击** 推进剧情与旁白
- 在特定时刻融入 **简单的平台跳跃玩法** 与**场景互动**
- 相应玩法操作有 **对应按键提示与机制说明**

## 特色亮点

### 🎨 美术与动效
- 独特的视觉风格与细腻的画面表现
- 精心设计的动态效果，增强沉浸感

### 🎵 音乐与音效
- 原创音乐与情境音效

### 🌉 玩法创新
- 围绕"桥"的概念设计交互机制，例如：
  - **引力桥**：利用引力到达终点
  - **失败生成桥**：失败会让成功更加容易

## 下载与运行

1. 前往**发行版**下载最新版本
2. 解压压缩包
3. 运行 `桥想.exe` 即可开始游戏
